# MuTools - Conversor de Imagens para Mu Online

## O que é o MuTools feito pelo DEV-Frostty?
O MuTools é uma ferramenta profissional desenvolvida especificamente para desenvolvedores de servidores privados de Mu Online Season 18. Permite converter imagens entre os formatos nativos do jogo (OZT, OZJ, TGA) e formatos editáveis (PNG, JPG) de forma rápida, segura e com preservação total da qualidade.

## O que o MuTools faz?

O aplicativo converte imagens do Mu Online para formatos editáveis e vice-versa, permitindo que você edite texturas, interfaces e elementos visuais do jogo usando ferramentas como Photoshop, GIMP ou qualquer editor de imagens.

### Facilidade de Uso

### Profissionalismo
- Interface moderna e limpa

## Funcionalidades Principais

### Conversão de Formatos
- PNG para TGA e vice-versa
- PNG para OZT (criar texturas do jogo)
- OZT para PNG (extrair texturas para edição)
- OZJ para JPG (extrair imagens JPEG)
- JPG para OZJ (criar imagens para o jogo)
- OZT para TGA (formato intermediário)

### Visualização Avançada
- Preview em tempo real de qualquer arquivo
- Zoom e pan para examinar detalhes
- Informações completas (dimensões, tamanho, nome)
- Controles intuitivos tipo Photoshop

### Conversão em Lote
- Selecione múltiplos arquivos de uma vez
- Converta todos para o mesmo formato
- Escolha a pasta de destino uma única vez
- Processamento automático e organizado

### Gerenciamento de Arquivos
- Lista todos os arquivos da pasta automaticamente
- Busca rápida por nome
- Filtro por tipo de arquivo
- Thumbnails para identificação visual
- Atualização da lista sem fechar o aplicativo

### Performance Otimizada
- Carrega arquivos grandes sem travar
- Cache inteligente para acesso rápido
- Processamento em segundo plano
- Suporta até 20MB por arquivo

## Casos de Uso

### Editar Texturas do Jogo
1. Converta OZT para PNG
2. Edite no Photoshop ou seu editor preferido
3. Converta PNG de volta para OZT
4. Substitua no jogo e veja as mudanças

### Personalizar Interface
- Extraia imagens de interface (OZJ) para JPG
- Edite ícones, botões e elementos visuais
- Converta de volta para OZJ
- Aplique suas personalizações

### Preparar Assets
- Converta suas texturas PNG para OZT
- Crie imagens JPEG no formato do jogo (OZJ)
- Organize e converta múltiplos arquivos de uma vez
- Mantenha compatibilidade total com ferramentas oficiais

## Requisitos

- Windows 10 ou superior
- Apenas 100MB de espaço em disco
- Não requer instalação de programas adicionais
- Versão portátil disponível (não precisa instalar)

## Formatos Suportados

- **OZT**: Texturas comprimidas do Mu Online
- **OZJ**: Imagens JPEG comprimidas do Mu Online
- **TGA**: Formato intermediário de texturas
- **PNG**: Formato padrão para edição
- **JPG/JPEG**: Formato padrão para imagens
